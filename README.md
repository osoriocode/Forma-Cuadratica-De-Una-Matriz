Aplicación web 100% frontend que permite generar la forma cuadrática de una matriz y determinar su definitud usando el criterio 
de Sylvester. Incluye validaciones avanzadas, autocompletado, corrección automática de simetría y diseño responsivo.

100% frontend web application that generates the quadratic form of a matrix and determines its definiteness using Sylvester’s 
criterion. It includes advanced validations, smart autocompletion, automatic symmetry correction, and a fully responsive 
design.
